# Lovttery Store Chatbot
## General 
This is a demonstrating chatbot built by using Google Generative AI Studio prompt design and Node interface.
You can ask questions about this Lovttery store which answers customers' questions.  

## Installation
1. Npm install
2. Create .env file and add your Gemini API key as:
     API_KEY="Paste API Key here"
3. Run node server.js
4. Open server port and interact with the bot.
